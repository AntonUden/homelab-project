Homelab project

I have been hosting servers on a small scale for a long time now but it all changed in early 2019 when i got my first real piece of server hardware.

I was looking for old server hardware on a site called blocket and i found a listing for an old hp server with 12 cpu cores and 32gb of ddr2 ram for around 50$, at this time i did not have much experience with real server hardware and it might not have been the best server to buy due to how old it was but it was still a very valuable resource for me to use while learing about servers.

After getting the server i started looking after a good operating system to use and since i wanted to run a bunch of virtual machines i ended up using Proxmox VE for the servers operating system.
After installing the operating system and getting getting familiar with proxmoox ve i set up my first vm that ran an apache2 web server and a mysql database replacing my old hp workstation that i used before getting this server, later on i also set up a minecraft server for me and my friends to play on

Around the same time as i got my first real server i also found a reddit community called r/homelab where people would post about their own home server projects.

Curent state of the project
